# Parity-plane mechanism

Status: public support note for the Shu-Durer paper.  This is a scoped finite
mechanism, not a universal invariant theorem and not a table-free symbolic
proof.

The mechanism explains why endpoint 24 is structurally visible in the binary
Durer layer.  In zero-based labels, a four-cell affine plane is bit-balanced
when each of the four coordinate bits occurs exactly twice.  Such a plane has
integer label sum

```text
2*(1+2+4+8) = 30.
```

## The NW parity plane

Use zero-based row and column coordinates, and flatten cells in row-major order.
The NW parity plane is the even-row/even-column four-cell set with row-major
indices

```text
0, 2, 8, 10.
```

In coordinates `(r1,r0,c1,c0)`, its directions are the row-parity and
column-parity directions.

## Public mechanism

1. **Affine-plane sum.**  An affine four-plane `{o,o+u,o+w,o+u+w}` has
   zero-based label sum `30` if and only if `u OR w = 15`.
2. **Visible B11 split.**  On the NW parity plane, `xor(labels)=B11`.  Since
   bit-balance forces xor zero, every case with `B11 != 0` is excluded.
3. **Residual forms.**  After the visible split, the normalized parity-plane
   reduction supplies the two transpose-symmetric row/column forms checked
   here, with `v in {2,4,6,8,12,14}`.  This note checks those residual
   cases; it does not derive a table-free classification of all residual forms.
4. **Finite residual check.**  For the row form, the residual direction
   constraints, the two diagonal constraints, and bijectivity force `a=v` and
   force `g` to miss the critical bit `nu2(v/2)`.  Hence the NW directions
   `a,g` do not cover all four bits, so the NW parity plane is not
   bit-balanced.  The column form follows by transpose symmetry.

The finite residual check is generated by
`scripts/analyze_parity_plane_mechanism.py` and recorded in
`results/parity_plane_mechanism.json` and
`results/PARITY_PLANE_MECHANISM_REPORT.md`.

## Guardrails

- The final residual step is a six-value finite check, not a table-free symbolic proof.
- The six residual values are scoped input cases from the normalized parity-plane reduction, not newly derived here.
- The mechanism is scoped to the normalized parity-plane model used in the
  paper.
- This note does not claim that 24 is a universal magic-square invariant.
